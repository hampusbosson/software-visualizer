# Software Visualizer frontend

React-frontend byggd med Vite, TypeScript, Tailwind CSS och Three.js via React Three Fiber.

## Utveckling

```bash
npm install
npm run dev
```

Öppna sedan adressen som Vite visar, normalt `http://localhost:5173`.

## Kommandon

- `npm run dev` – startar utvecklingsservern
- `npm run build` – typkontrollerar och skapar en produktionsbuild
- `npm run lint` – kör ESLint
- `npm run preview` – förhandsvisar produktionsbuilden

Three.js-komponenter ligger i `src/components/three`.
